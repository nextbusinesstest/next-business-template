This is a starter template for Next Business Agency. It's designed to be cloned and rebranded quickly for each client.

How to run locally:
1. npm install
2. npm run dev
3. Open http://localhost:3000

Deploy:
- Connect repository to Vercel for zero-config deployment.

Automating client site creation:
- See /scripts/clone_and_brand.js: a simple Node script that copies the template directory, replaces strings (NEXT_BUSINESS -> ClientName, colors) and prepares a ready-to-push project folder.
